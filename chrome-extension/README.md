# Pipelio Chrome Extension

Save jobs from any job board to your Pipelio pipeline with one click.

## Supported Job Boards
- LinkedIn
- Indeed
- Glassdoor
- Lever
- Greenhouse
- Workday
- Ashby
- ZipRecruiter
- Monster

## Install for Development (Load Unpacked)

1. Open Chrome → go to `chrome://extensions`
2. Enable **Developer mode** (top right toggle)
3. Click **Load unpacked**
4. Select this `chrome-extension` folder
5. The Pipelio icon appears in your toolbar

## Connect to Your Account

1. Click the Pipelio icon in Chrome
2. Enter your Pipelio URL (e.g. `https://yourapp.vercel.app`)
3. Go to **Pipelio → Settings** → generate an API token
4. Paste the token into the extension
5. Done — visit any job listing and save it instantly

## Publish to Chrome Web Store

1. Zip the entire `chrome-extension` folder
2. Go to https://chrome.google.com/webstore/devconsole
3. Pay the one-time $5 developer fee
4. Upload the zip → fill in store listing → submit for review
5. Review takes 1–3 business days

## Icons

Replace the placeholder icons in `/icons` with real PNG files:
- `icon16.png` — 16×16px
- `icon48.png` — 48×48px  
- `icon128.png` — 128×128px

Use the Pipelio logo (sage green briefcase on white/transparent background).
